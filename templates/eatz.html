Eatz

*Still in development could contain bugs

Eatz is an application I built for my girlfriend to be able to finally decide what to eat. It includes a simple form that collects data and will output the resturants nearby that fit the qualifications specified. The data is pulled in through the Google Developers API and uses real time data to provide accurate results. My favorite feature is the "Ive asked Alex what he wants" which will choose a single resturant based on what I would like, making the choice easy!


Tech Stack:

HTML, CSS, Javascript, Python, Heroku Pipelines, Google Developer APIs
